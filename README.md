# WeTunes
WeTunes là một trang mạng xã hội âm nhạc dành cho mọi lứa tuổi, mọi tầng lớp. Là nơi mà mọi người có thể kết nối với nhau thông qua những giai điệu được sưu tầm và chọn lọc.
> Not iTunes, be WeTunes
